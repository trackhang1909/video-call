# dev
~~~
**** YÊU CẦU ****
- MongoDB Community Server (https://www.mongodb.com/try/download/community).
- Node.js phiên bản từ 14.17.0 (https://nodejs.org/dist/v14.17.3/node-v14.17.3-x64.msi).
- NPM phiên bản từ 6.14.13.

**** THỰC HIỆN ****
- Giải nén tệp source code chương trình vào một thư mục bất kỳ (Ví dụ: D:\project\src)
- Mở chương trình Command Prompt và dùng lệnh cd để di chuyển đến thư mục chứa source code.
	Ví dụ:
	Source code được chứa trong thư mục D:\project\src.
	Sử dụng lệnh: 'cd /d D:\project\src' để di chuyển đến thư mục.
- Tiếp tục dùng lệnh 'npm install' để cài đặt các module cần thiết cho chương trình.
- Dùng lệnh 'npm start' để khởi động server cho chương trình.
- Sau khi lệnh 'npm start' chạy xong, bây giờ có thể truy cập chương trình bằng cách vào trình duyệt web và truy cập đường dẫn https://localhost:8000.

**** LƯU Ý ****
- Các lệnh sử dụng được đặt trong dấu ngoặc đơn.
- Các bước được thực hiện trong môi trường Windows 10. Các hệ điều hành khác có thể có sự thay đổi.
~~~
